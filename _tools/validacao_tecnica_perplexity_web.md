# Validação Técnica com Perplexity e Web

## Próxima tarefa

Validar tecnicamente os conteúdos já produzidos de **Física** e **Química**, especialmente a **unidade 4**, usando:

- Perplexity MCP (`perplexity`, modelo `sonar`);
- busca web complementar quando necessário;
- fontes confiáveis e atuais.

## Escopo inicial

- `Fisica/*/unidade-4*/capitulo_*.md`
- `Quimica/*/unidade-4*/capitulo_*.md`

## O que validar

- conceitos centrais;
- fórmulas, unidades e relações matemáticas;
- nomes de cientistas, datas e atribuições históricas;
- limites didáticos do assunto por ano/série;
- exemplos e aplicações práticas;
- possíveis simplificações excessivas ou afirmações imprecisas.

## Como registrar

Para cada disciplina, produzir um relatório curto com:

- capítulo revisado;
- ponto validado;
- fonte usada;
- correção necessária, se houver;
- status: `ok`, `ajustado` ou `pendente`.

Se houver correção, ajustar o Markdown e fazer commit/push ao final do lote validado.
