## 2. Abertura

- **H1**: `# Capítulo X — [Tema]`
- **H2 direto**: pergunta-gancho como heading. **SEM** o rótulo `## Pergunta-problema` + `*pergunta?*`.
- **Parágrafo narrativo**: cena → tensão → ponte.
  - Fund 1: 3–4 frases
  - Fund 2: 4–5 frases
- **Box 💭 Pense um pouco**: pergunta provocativa pra ativar curiosidade.

---

## 3. Padrão dos subtópicos (N.1, N.2)

Cada subtópico = **1–2 frases de definição + conector + bullets**:

```
**[Termo]** é [definição direta]. [Frase de ponte opcional]. [Conector terminando em :]

- bullet 1;
- bullet 2;
- bullet 3 (último com .).
```

**Regras:**
- Definição em **negrito inline** — NÃO usar "podemos definir" / "chamamos de"
- 1–2 frases curtas antes do conector
- Bullets paralelos (mesma classe gramatical)
- Bullets com `;` no fim (último com `.`)
- **Número de bullets é flexível** — 1, 2, 3, 4 ou 5 conforme o conteúdo. Não travar em 3.
- **Subtópico sem lista**: máx 2 frases de prosa direta

---

## 4. Boxes — catálogo e formato

**Formato obrigatório (sempre blockquote):**

```
> **emoji Título**
>
> Conteúdo.
```

**Catálogo:**

| Emoji | Título | Quando usar |
|---|---|---|
| 💭 | Pense um pouco | Abertura — pergunta provocativa |
| 💬 | Para Conversar | Final do bloco bíblico — pergunta de reflexão |
| 💡 | Você sabia? | Curiosidade factual (ex.: cientista histórico) |
| 🔬 | Ciência do Dia a Dia | Conexão com cotidiano |
| ⏸️ | Pare e Pense | Pausa reflexiva |
| 📏 | Medidas Interessantes | Dado quantitativo |

**Distribuição:** ~1 box por seção numerada (4 boxes no corpo) + 💭 na abertura + 💬 no fechamento bíblico = ~6 boxes totais.

---

## 5. Blockquote-respiro (sem emoji)

Parágrafos de **ressalva, nuance ou aviso** que vêm depois de uma afirmação principal viram blockquote simples (`> Texto.`) para dar respiro visual.

**SIM** converter (exemplos):
- "Reversibilidade ajuda a identificar, mas não é regra absoluta."
- "Irreversível nem sempre é químico, e reversível nem sempre é físico."
- Avisos de segurança ("sentir cheiro nunca deve envolver...")

**NÃO** converter:
- Definições
- Introduções de seção
- Frases-conclusão diretas

## 6. Fechamento (ordem fixa)

### 6.1 `## O que a Bíblia diz sobre [tema]`

Padrão final (validado no POC Fund 2):

```
## O que a Bíblia diz sobre [tema]

> *"versículo"* — **Referência**

[1 parágrafo de 2 frases — cena bíblica curta + ponte pra ciência, terminando em conector ":"]

- bullet 1 do princípio aplicado;
- bullet 2 do princípio aplicado.

> **💬 Para Conversar**
>

```

**Síntese máxima**: ~30 palavras de prosa + 2 bullets curtos + 1 pergunta. SEM longas narrativas bíblicas.

### 6.2 `## Simplificando`

**1 frase única** de resumo do capítulo. SEM segundo parágrafo. SEM bullets.

### 6.3 `## Para não esquecer`

3–4 bullets em formato glossário:

```
- **termo:** definição curta;
- **termo:** definição curta;
- **termo:** definição curta;
- **termo:** definição curta.
```

## 7. Calibração por faixa

### Fund 1 (4º e 5º ano)

- **Frases**: 8–12 palavras
- **Bullets**: 1–2 quando possível; quando 3+ itens são essenciais, usar prosa inline ou bullets ultra-curtos
- **Vocabulário**: cotidiano puro
- **Personagens**: crianças (ex.: Marina, Lucas) na narrativa de abertura
- **Boxes típicos**: 💭, 🔬 (cotidiano), 📏 (números pequenos), ⏸️
- **POC referência**: `Ciencias/5ano/unidade-4-corpo-humano/capitulo_02_circulacao-e-distribuicao.md`

### Fund 2 (6º, 7º, 8º, 9º ano)

- **Frases**: 12–20 palavras
- **Bullets**: flexível — 2, 3, 4 ou 5 conforme o conteúdo pede
- **Vocabulário**: técnico com definição inline em **negrito**
- **Personagens**: cientistas históricos (em 💡 Você sabia?)
- **Boxes típicos**: 💭, 🔬, ⏸️, 💡 (histórico), 📏
- **POC referência**: `Ciencias/6ano/unidade-4-transformacoes-da-materia/capitulo_01_transformacoes-fisicas-e-quimicas.md`

## 8. Regras de preservação

**NUNCA alterar:**
- Conceitos e definições científicas do original
- Ordem das seções (a menos que claramente fora de progressão)
- Personagem escolhido pelo autor
- Versículo escolhido pelo autor
- Vocabulário técnico apropriado para a idade

**PODE ajustar:**
- Frases muito longas → encurtar
- Listas em prosa inline → bullets
- Boxes em formato antigo (`> 💭 **Pense um pouco:**`) → formato novo
- Section headings em Title Case → sentence case
- Mais de 2 subseções por seção → consolidar em 2
- Parágrafos de ressalva → blockquote-respiro

---

## 9. Regras de adição

**SEMPRE adicionar se faltar:**
- Glossário com 3–4 bullets em "Para não esquecer"
- Box `💬 Para Conversar` no final do bloco bíblico
- Box em seção que estiver sem (~1 box por seção)

**NUNCA inventar:**
- Versículos novos (manter os escolhidos pelo autor)
- Dados quantitativos não presentes no original
- Personagens não mencionados no original

## 10. Transformações comuns (mapa)

| Antes | Depois |
|---|---|
| `## Pergunta-problema\n\n*Pergunta?*` | `## Pergunta?` (H2 direto) |
| `> 💭 **Pense um pouco:**  \n> texto` | `> **💭 Pense um pouco**\n>\n> texto` |
| `## 1. Título Em Title Case` | `## 1. Título em sentence case` |
| 3 subseções (3.1, 3.2, 3.3) | 2 subseções consolidadas |
| "Podemos definir X como..." | "**X** é..." |
| "Chamamos de X..." | "**X** é..." |
| Parágrafo de ressalva | Blockquote sem emoji |
| `**Olhe como você pode fazer:**` + bullets | Bloco bíblico no formato novo |
| 🤝 Sua Parte | Remover completamente |
